# biswas-shabel-2026
These folders contain code used to generate line and bar graphs in Figures 1-4, S1, and S2 in Matlab. They are mostly variants of the same program that filters the dataset and makes line and bar plots for the desired time bins. Time bins are 100 ms long and the first 10 seconds (bins 1-100) are the pre-odor baseline periods. Data were filtered for learners or non-learners by setting a lick discrimination threshold (i.e., lick_discrim_thresh = .01). If the threshold were 1, then it would not be applied. Two helper functions are used to filter the data: one for learners and one for nonlearners. The files also contain the code for the linear mixed effect models used for statistical analysis.
